# Personal Assistant

A Python-based personal assistant that can:
- Read, draft, and send emails
- Get top HackerNews stories
- Access MLB data and games

## Features
- **Email:** Manage your inbox and send drafts
- **HackerNews:** Fetch the latest headlines
- **MLB:** View schedules and game info
- Streamlit UI is incorporated in this project.

---
A lightweight tool for productivity, news, and sports.
